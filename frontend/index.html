TRACENET · B.E. ISE IV · RVCE

Resistance Moves Like a Graph

Bacteria spread antibiotic resistance horizontally — via plasmids that jump between species during conjugation. This network models 16 bacterial species as nodes and 144 directed transfer pathways as weighted edges, with weights derived from Jaccard similarity of plasmid-borne ARG repertoires corrected by taxonomic distance.

Eight classical DAA algorithms are applied in sequence to trace spread, detect community structure, find high-risk paths, and determine optimal containment.

Breadth-First Search

Mapping the Spread

Starting from Klebsiella pneumoniae (46 plasmid ARGs), BFS discovers every species reachable via horizontal gene transfer. The algorithm uses a FIFO queue to process nodes layer by layer, assigning hop-distances that define the spread frontier. O(V + E).

Watch the propagation wave. Four species receive no color — they belong to a structurally separate Gram-positive cluster that is unreachable from any Gram-negative starting point.

4isolated (Gram+ cluster)
1max hop distance

Cross-Gram Jaccard × τ(0.5) falls below MIN_WEIGHT=0.05 for all pairs. No edges exist between Gram-negative and Gram-positive clusters — biologically consistent with ARG transfer barriers across membrane structure boundaries.

Kosaraju's Algorithm

Finding Resistance Bubbles

A Strongly Connected Component is a maximal group where resistance can circulate indefinitely — every species can reach every other. Kosaraju's algorithm finds all SCCs in two DFS passes: one on the original graph (computing finish order), one on the transposed graph (identifying components). O(V + E).

Two bubbles emerge. Within each, resistance recirculates without escape. The structural isolation is a direct consequence of the Gram classification threshold.

Component A: K. pneumoniae, E. cloacae, P. aeruginosa, A. baumannii, E. coli, S. enterica, K. oxytoca, C. freundii, P. mirabilis, S. marcescens, A. pittii, P. putida. Component B: E. faecium, S. aureus, E. faecalis, C. jejuni.

Kahn's Algorithm

Gene Acquisition Order

Resistance genes have clinical co-occurrence dependencies: isolates carrying CTX-M typically already carry TEM. Kahn's topological sort finds a valid ordering on the 10-node ARG dependency DAG by repeatedly selecting zero-in-degree nodes. O(V + E).

The inset shows the ARG DAG. Kahn's algorithm peels nodes in valid topological order — the sequence a pathogen would realistically accumulate these resistance families. A cycle detection check asserts DAG validity.

Order: tetM → sul1 → blaTEM → blaSHV → blaCTX-M → blaOXA-48 → blaNDM-1 → mcr-1. The DAG is valid. Kahn's output size equals |V|, confirming acyclicity.

Boyer-Moore

Scanning for NDM-1

Given a reference ARG sequence from CARD v4.0.1, Boyer-Moore scans for a 30bp query primer using the bad-character heuristic — scanning the pattern right-to-left, then jumping forward based on the mismatch character's last occurrence. O(n/m) average on a 4-character DNA alphabet.

Each upward jump in the inset animation is a position the naive algorithm would have checked but Boyer-Moore skips entirely. The advantage compounds with longer sequences.

Dijkstra's Algorithm

Highest-Risk Spread Path

Edge weights are transfer probabilities w ∈ (0,1]. Dijkstra converts them internally to −log(w) distances — high probability becomes short distance, making the shortest path the most probable route. A min-heap priority queue drives optimal expansion order. O((V+E) log V).

The golden path traces the highest-probability route from K. pneumoniae. The highest-risk pair in the entire network is then revealed — within the Gram-positive cluster, where same-genus taxonomy maximises transfer probability.

71.4%peak transfer probability
E. faecium → E. faecalishighest-risk pair

Jaccard = 0.714, τ = 1.0 (same genus Enterococcus). The two Enterococcus species share the most ARG families of any pair in the network — a direct, high-probability horizontal gene transfer channel.

06 / 08 · Unit IV

Floyd-Warshall

The Vulnerability Matrix

Floyd-Warshall computes all-pairs shortest −log(w) distances in three nested loops, exposing every species' vulnerability to every potential source simultaneously. O(V³). For V = 16: 4,096 operations — negligible.

The heatmap encodes spread distance: red cells are close (high probability path), blue cells are distant, black cells unreachable. The species with the lowest average incoming distance is the most systemically vulnerable.

K. pneumoniaemost vulnerable species
4 096distance operations

K. pneumoniae receives from the most species with the shortest average −log distance. Its high ARG count (46 plasmid genes) makes it simultaneously the densest spreader and the most accessible target in the Gram-negative cluster.

07 / 08 · Unit IV

Greedy Containment

Approximate Intervention

The containment problem — remove minimum edges to isolate environmental reservoirs from ESKAPE targets — is NP-hard in the multi-source case (reducible from Minimum Vertex Cover). A greedy heuristic provides a polynomial approximation: remove edges in descending weight order until sources cannot reach targets.

The greedy algorithm is globally blind — it removes high-weight edges throughout the network even when they are not on any source→target path. This produces a valid but highly suboptimal solution, demonstrated by comparison with B&B.

39edges greedy removed (hospital subgraph)
NP-hardexact multi-source problem

On the hospital subgraph (10 nodes): greedy removed 39 of 42 edges before the containment condition was satisfied. Most removed edges were irrelevant — they lay entirely within the Gram-negative cluster, disconnected from the actual threat path.

08 / 08 · Unit V

Branch and Bound

Exact Containment

On the 10-node hospital subgraph, Branch and Bound performs exhaustive search with intelligent pruning. It initialises an upper bound from the greedy solution, then prunes any branch where the removed set size already equals or exceeds the best known cost. O(2E) worst case — fast in practice.

The result is the provably optimal minimum cut: the four direct edges from environmental sources to ESKAPE targets. Every path from source to target passes through exactly these four edges — they are the minimum s-t cut.

4edges in optimal cut (B&B)
39edges greedy removed
9.75×improvement over greedy

Optimal cut: E. faecalis→E. faecium, E. faecalis→S. aureus, C. jejuni→E. faecium, C. jejuni→S. aureus. These four direct source→target edges constitute the minimal intervention. B&B proves optimality; greedy cannot.

SUMMARY

Eight Algorithms, One Network

TraceNet demonstrates that antibiotic resistance spread is fundamentally a graph problem — and that the classical techniques across DAA Units II–V are precisely the tools needed to analyze it rigorously.
